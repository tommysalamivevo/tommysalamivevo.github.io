Bite of the Mad Dog
A modification of the game "Yakuza 3" that changes the main character to Goro Majima! This mod also adds 6 fully-fledged movesets, 5 lite movesets, and 40+ heat actions. The interface, color grading, and voicelines are overhauled as well.
More Heat Auras
This is a modification of the game "Yakuza 0" that ports over 8 heat auras from the game "Like a Dragon: Ishin!" over the 8 player-controlled auras. The colors of the auras have deeper, more saturated colors while the particles are now fire. Recolored HUD elements are included.
Embedded Xinu
A reimplementation of the original Xinu operating system. Embbeded Xinu allows for several ports to the MIPS processor, linksys routers, Rpie's and RiskV. Additionally its lightweight design makes it suitable for wireless routers.
Melee Lucina
Modifies the game "Super Smash Brothers Melee" to add Lucina from Fire Emblem Awakening as a playable character. Her moveset is Marth's but has been altered to mirror her appearance in later games. Custon Character Select Screen and Stock icons are included.
Camilla
This project is a modification of the game "Payday 2" to add the character Camilla from Fire Emblem Warriors to Payday 2 using Legendary Armour Skins framework. Includes third person models, first person arm models, and menu icons.
Dungeon Defenders
This is a physics-based tower defense game developed in Unity for release on desktop and mobile platforms. The game contained 5 levels of predetermined waves of enemies that players must survive through balancing the economy and cooldowns.
